# RythuMowa Storefront Design System

## Color Palette
```css
--primary-green: #378365
--primary-green-hover: #2d6b52
--accent-orange: #ffa96e
--accent-yellow: #fff0d1
--text-primary: #1e363e
--text-secondary: #111111
--text-muted: #666666
--bg-gradient: linear-gradient(181deg, rgba(249,244,236,1) 0%, rgba(255,255,255,1) 100%)
```

## Typography
- Primary Font: 'Montserrat', sans-serif
- Logo Font: 'Segoe Print', cursive
- Weights: 400 (medium), 600 (semibold), 700 (bold)

## Component Styles

### Header
- Logo: 127x127px
- Search bar: Rounded 40px, border #7f4000, shadow
- Cart icon with item count
- Navigation: Montserrat medium, 18px, tracking 1.71px

### Product Card
- Image: 196px height, rounded 5.4px
- Star rating: 5 stars, 24px each
- Price: Bold, 17px
- Discount badge: Red #fc3333
- "From" text: Green #7fb750
- Product name: 16px, min-height 67px

### Section Headers
- Background: #fff0d1
- Height: 97px
- Title: Montserrat semibold, 30px
- Shadow: 4px 4px 4px #dea40180

### Promotional Banners
- Heights: 180px - 262px
- Rounded: 5.34px
- Colors: #cfdaf1 (blue), #fdcf99 (orange), #d7ffb6 (green), #ffecc4 (yellow)

### Buttons
- Primary: #378365, rounded 48px
- Hover: #2d6b52
- Font: Montserrat bold, 14px

### Footer
- Background: #378365 (green section)
- Payment icons: 41px height
- Links: Montserrat medium, 14px, color #666666

## Layout
- Max width: 1307px - 1312px
- Padding: 16px (px-4)
- Grid gaps: 24px - 32px
- Section spacing: 64px (mt-16)

## Product Grid
- Year End Sale: 4 columns
- Products on Sale: 6 columns
- Latest Products: 6 columns
- Gap: 24px - 32px

## Images
All product images should be:
- Aspect ratio: ~1:1 to 4:3
- Format: WebP or PNG
- Optimized for web
- Lazy loaded

## Badges
- "SOLD OUT": Red #ff2121, top-right
- "Organically Grown": Green #9bd967, bottom-left
- Size badges: Green #9bd967, 23.4px font

## Shadows
- Cards: shadow-md to shadow-xl on hover
- Search bar: 5px 5px 10px #7f400099
- Section headers: 4px 4px 4px #dea40180
